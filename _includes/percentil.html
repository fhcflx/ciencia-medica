Percentil (_estatística_) neste contexto é uma medida estatística que indica a posição relativa de um valor dentro de um conjunto de dados ordenados. O percentil é o valor abaixo do qual uma certa porcentagem dos dados cai. Por exemplo, o 50º percentil (ou mediana) é o valor abaixo do qual 50% dos dados estão localizados. Se um aluno está no 90º percentil em uma prova, isso significa que ele obteve uma pontuação melhor do que 90% dos outros alunos que fizeram a mesma prova. O 10º percentil (ou percentil 10), por outro lado, indica que 10% dos dados estão abaixo desse valor. Os percentis são frequentemente usados para entender a distribuição de dados e identificar valores atípicos ou extremos. Eles são úteis em diversas áreas, como educação, saúde e finanças, para comparar desempenhos ou características entre diferentes grupos ou populações.
