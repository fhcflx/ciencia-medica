O tema Tufte-Jekyll é a extensão natural do trabalho feito por Edward Tufte e seus colaboradores no Github, os quais criaram um arquivo CSS que permite a desenvolvedores web utilizar o mesmo estilo simples e elegante empregado nas suas publicações.

A fim de incorporar esets estilos em um tema Jekyll, eu fiz pequenas modificações com o intuito de manter o espírito dos estilos CSS no repositório Github.

Note que este é um layout que usa toda a largura da página. Isto foi configurado incluindo layout: full-width no arquivo YAML front matter desta página. Lembre que as funcionalidades de notas laterais e de margem não funcionarão neste layout!

Você pode encontrar o código fonte do Jekyll em github.com/jekyll/jekyll.
